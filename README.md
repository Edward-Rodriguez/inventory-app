## Technologies

Project is created with:

- TypeScript version 4.0.2
- CSS
- Semantic UI
- [Angular CLI](https://github.com/angular/angular-cli) version 10.0.7

## Setup

To run this project, clone this repo and  
install it locally using npm:

```
$ cd ./inventory-app
$ npm install
$ npm start
```
